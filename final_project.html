Final Project.

The program my partner and I have created is called Extreme Tag. We used object orientation to organize and execute our code for this game application. The goal of this game is to tag your teammate as many times as possible before the blocks run out, as well as simultaneously avoiding the blocks that move from left to right and from top to bottom. Users can use their mouse cursor to move the bitmoji around to avoid the blocks. To tag a teammate, move the mouse cursor near the teammate to tag them. Each time the user has tagged the teammate, the speed of the blocks increases. If the user is hit by a block, 5 points will be deducted from their current score and the block they were hit by will turn dark blue. If the user was able to tag their teammate 15 times before all the blocks run out, then the user has finished the game and a “Game Over” text will be displayed on the screen along with their score and the number of tags.
